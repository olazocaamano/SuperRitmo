# El Super Ritmo - Official Website

## Project Overview
This repository contains the source code for the official web platform of El Super Ritmo, a professional tropical music ensemble. The project is designed as a high-performance, responsive single-page application focused on branding, user engagement, and lead generation through direct communication channels.

## Technical Stack
*   **Frontend:** Semantic HTML5, CSS3 (Custom Properties and Variables).
*   **Typography:** Montserrat via Google Fonts API.
*   **Interactivity:** Vanilla JavaScript (ES6+).
*   **Background Rendering:** HTML5 Canvas API for dynamic particle systems.
*   **Deployment:** Vercel (Continuous Integration and Continuous Deployment via GitHub).

## Key Features
*   **Dynamic Particle System:** A custom-coded background utilizing the Canvas API to create a fluid, rhythmic visual environment that aligns with the musical theme.
*   **Responsive Navigation:** A fully optimized mobile menu system including a JavaScript-driven hamburger toggle and glassmorphism design patterns.
*   **Lead Generation Integration:** Optimized Call-to-Action via WhatsApp Business API integration, featuring pre-formatted messaging for professional inquiries.
*   **High-End UI and UX:** Implementation of neon aesthetics, CSS backdrops, and smooth transitions to ensure a modern and immersive user experience.

## Installation and Local Development
To run this project locally, follow these steps:

1. Clone the repository:
   git clone https://github.com/olazocaamano/SuperRitmo.git

2. Navigate to the project directory:
   `cd SuperRitmo`

3. Open the project:
   Launch index.html using a local development server such as Live Server in VS Code to view the project at localhost.

## Performance Optimization
*   **Asset Management:** Images are handled through CSS properties to maintain aspect ratios and visual integrity across all viewports.
*   **Code Efficiency:** Minimal use of external libraries to ensure low latency and optimized loading times.
*   **Responsive Breakpoints:** Targeted media queries at 768px to ensure seamless transitions between mobile, tablet, and desktop layouts.
